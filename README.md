# HP Omen 15 dc000xxx with Open Core 0.6.3 Hackintosh
Works with Big Sur 10.16


<i>What works:</i>

* Battery, battery status
* Wi-fi (not full speed but is ok)
* Bluetooth
* Audio
* Graphics Intel
* SD card reader
